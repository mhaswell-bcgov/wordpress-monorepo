# BC Government WordPress Search Plugin

A WordPress plugin that provides advanced search functionality with a comprehensive suite of custom blocks for building flexible, accessible search experiences.

## Key features:
### Blocks
The following search-related blocks are available for use in the page editor.
See the individual page for each block to learn more about them:
- [Search Bar](guide/Blocks/Search): Main search interface for querying documents and posts.
- [Search Active Filters](guide/Blocks/SearchActiveFilters): Show currently applied filters and allow users to clear them.
- [Search Modal](guide/Blocks/SearchModal): Modal-based search overlay for compact implementations.
- [Search Post Filter](guide/Blocks/SearchPostFilter): Dynamic filtering by custom post types and metadata.
- [Search Result Count](guide/Blocks/SearchResultCount): Display the total number of search results.
- [Search Results Post Metadata Display](guide/Blocks/SearchResultsPostMetadataDisplay): Display custom metadata alongside search results.
- [Search Results Sort](guide/Blocks/SearchResultsSort): Allow users to reorder search results by various fields.
- [Search Taxonomy Filter](guide/Blocks/SearchTaxonomyFilter): Filter results by taxonomies (categories, tags, etc.).
