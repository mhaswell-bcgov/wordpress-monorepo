# Search Results Sort Block

Enables users to reorder search results by various fields and sort directions.
![`Search Results Sort`:](/images/SEARCH_RESULTS_SORT.png)
## Features
- Sort by title, date, or custom metadata fields
- Ascending/descending sort options
- Select default sort field and order

## Settings
- **Sort Field**: Choose which metadata field to sort by
- **Sort Order**: Select sort direction (newest, oldest, asc, desc)

## Usage
1. Add the Search Results Sort block above or below your search results.
2. Configure sort field and order in the block settings panel.
3. Users can change sort order on demand.
