# Search Block

Provides a search input field for querying posts and documents.
![`Search Bar`:](/images/SEARCH_BAR.png)

## Features
- Simple search bar for user queries
- Integrates with other search blocks for advanced filtering and results

## Settings
- No user-configurable settings

## Usage
1. Add the Search block to any page or template.
2. Optionally combine with filter, results, and metadata display blocks for a complete search experience.
