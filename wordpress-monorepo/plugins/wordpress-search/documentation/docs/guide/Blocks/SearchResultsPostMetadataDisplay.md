# Search Results Post Metadata Display Block

Displays custom metadata and taxonomy information for each search result.
![`Search Results Post Metadata Display`:](/images/SEARCH_RESULTS_METADATA.png)
## Features
- Shows custom post metadata fields
- Displays taxonomy terms and categories
- Customizable font size for metadata display
- Automatically excludes internal/version fields

## Settings
- **Font Size**: Set metadata text size

## Usage
1. Add the Search Results Post Metadata Display block below your search results.
2. Configure font size in the block settings panel.
3. Metadata displays automatically for each result.

## Excluded Fields
- `document_file_id`, `document_file_versions`, `document_file_url`, `document_file_name`, `document_file_size`, `document_file_type`, `inpage_nav`, `show_inpage_nav`, and any field prefixed with `_`.
