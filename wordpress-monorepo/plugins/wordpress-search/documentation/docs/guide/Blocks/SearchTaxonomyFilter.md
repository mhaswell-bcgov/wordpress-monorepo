# Search Taxonomy Filter Block

Enables filtering search results by WordPress taxonomies (categories, tags, custom taxonomies).
![`Search Taxonomy Filter`:](/images/SEARCH_TAXONOMY_FILTER.png)
## Features
- Multi-select taxonomy filtering
- Dynamic term lists
- Hierarchical taxonomy support

## Settings
- **Taxonomies**: Select which taxonomies users can filter by.

## Usage
1. Add the Search Taxonomy Filter block to your search results area.
2. Configure taxonomies in the block settings panel.
3. Users can select terms to filter results.
