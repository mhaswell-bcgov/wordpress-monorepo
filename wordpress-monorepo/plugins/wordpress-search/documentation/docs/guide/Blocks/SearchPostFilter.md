# Search Post Filter Block

Enables filtering search results by post type and custom metadata.
![`Search Post Filter`:](/images/SEARCH_POST_TYPE_FILTER.png)
## Features
- Multi-select post type filtering
- Customizable underline color for filter UI
- Dynamic filter options

## Settings
- **Post Types**: Select which post types users can filter by.
- **Underline Color**: Choose the color for the underline indicating the active filter.

## Usage
1. Add the Search Post Filter block to your search results area.
2. Configure post types and underline color in the block settings panel.
3. Users can select multiple filter options to narrow results.
